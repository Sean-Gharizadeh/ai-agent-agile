\# 🤖 AI Agent



Simple local AI agile agent built in Python.



\## 🚀 Features



\- Multi-command system

\- Chat logging

\- Modular architecture

\- Easy to extend



\## 📂 Project Structure



agent.py

app.py

main.py

commands.py

utils.py



\## 🛠️ Setup



Install dependencies: pip install -r requirements.txt



Run the agent: python main.py



\## 🧠 Description



This project is a lightweight AI agent that can process commands, log conversations, and be extended with additional functionality.
